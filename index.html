How it works

Three phases.
One precise brief.

Most search engines treat everyone the same. Systematic Search opens a dialogue first — then builds a response that fits you specifically.

01
Socratic Dialogue
A short conversation maps your expertise level, goals, and preferred way of receiving information. Usually 3–5 questions. Never more than 2 minutes.
02
Intelligent Synthesis
Gemini 2.5 Pro takes your profile and queries the live web — cross-referencing sources, weighing credibility, and identifying what actually matters to you.
03
Your Discovery Brief
A structured, narrative research brief calibrated to your level. Not a list of links — a document that reads like a colleague who knows your work wrote it for you.

Getting started

Running in
under 5 minutes

1
Clone the repository
Pull the project to your local machine using Git.
git clone https://github.com/lochanharishwar/SYSTEMATIC-SEARCH.git cd SYSTEMATIC-SEARCH
2
Install dependencies
Requires Node.js v18+. Install all project dependencies with npm.
npm install
3
Add your Gemini API key
Open .env.local and paste your key. Get one free at aistudio.google.com.
# .env.local GEMINI_API_KEY=your_key_here
4
Start the dev server
Launch the app locally. Go fullscreen for the best experience.
npm run dev # → http://localhost:5173
